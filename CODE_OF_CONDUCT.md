# Code of Conduct

TabRefDetect follows a simple research collaboration code of conduct.

## Expected Behavior

- Be respectful and constructive in issues, discussions, and pull requests.
- Focus feedback on the technical work, evidence, and reproducibility.
- Be patient with contributors who are new to the project or to agent workflows.
- Credit prior work, datasets, tools, and model providers appropriately.
- Handle research data, annotations, and credentials with care.

## Unacceptable Behavior

- Harassment, personal attacks, or discriminatory language.
- Publishing private datasets, credentials, or non-redistributable papers.
- Misrepresenting model outputs as human gold annotations.
- Intentionally submitting code that hides data leakage or undermines reproducibility.

## Reporting

If you see behavior that violates this code, please open a private security advisory if it involves sensitive data or credentials. For ordinary conduct issues, contact the repository maintainers through GitHub.

Maintainers may remove comments, close issues, or block contributors when necessary to keep the project safe and constructive.
