# TabRefError Dataset Placeholders

This directory is intentionally a placeholder in the public repository.

The full TabRefError dataset is not committed here because it may include original papers, extracted table content, human annotations, OCR outputs, or other files that require controlled distribution.

## Directory Layout

- `citekeypdf/`: placeholder for citing-paper PDFs.
- `originalpdf/`: placeholder for cited-paper PDFs.
- `originalkey/`: placeholder for cited-paper keys or metadata.

Only tiny placeholder files are tracked so that downstream scripts can refer to the expected directory structure.

## Access

Please see the repository README and GitHub Releases for information about any public subset. For full dataset access, contact the project maintainer.

## Contribution Rule

Do not submit private PDFs, full OCR outputs, private annotations, or generated model responses in pull requests.
